# Spotify Album Sales


Olá segue uma descrição breve das tecnologias usadas  para desenvolver a solução.
 * ASPNET Core
 * Entity Framework
 * Microsoft Web API
 * NSwag (Swagger) * SQL Server

 
 Para executar a solução observar a pasta ./Scripts , e executar os scripts nesta pasta para criar e inserir alguns dados no banco de dados. Após isso configurar a conexão do banco no arquivo appsettings.json na connectionString : SpotifyAlbumSalesDbContext. Após isso executar a aplicação.
 Acessar a url /swagger para acessar a documentação dos métodos.
